Introcution

Sugoï is a simple and unbloated web framework with very few dependencies.
It’s an opensource fork of the web framework used at Motion Twin since years. It means it’s scalable and production ready : millions of pages are served each month relying on this framework.
Most of the original code has been written by Nicolas Cannasse

Get started

Requirements : you need a webserver, mysql and neko or PHP.
I personnaly use mod_neko and apache prefork on my developement machine. How to setup this is not covered here.
Neko is the target of choice for this framework as it’s faster than PHP, a few features are not availble in PHP, and finally the exceptions and the call stack are more informative in neko than PHP (no source mapping in PHP?).

Install sugoi :
``bash haxelib git sugoi git@github.com:bablukid/sugoi.git

Clone the demo website
git@github.com:bablukid/sugoi-demo.git

create your vhost and your local host name, create a database in mysql, edit config.xml accordingly.
The demo site should be available in your browser !
